参数

参数 数据类型 默认值 描述
mode String 'horizontal' 定义slider滚动的方向，有三个值可供选择 'horizontal', 'vertical', 'fade'
slideSelector String '' 所选元素将作为一个滑块参与滑动；默认值为空，表示当前滑动容器的直接子元素都作为滑块参与滑动,任意jquery选择器，例如：’div.slide’
infiniteLoop boolean true 点击“下一步”，而上一个幻灯片将过渡到第一个幻灯片，反之亦然
hideControlOnEnd boolean false 假如设置为 true，“下一步”控制将隐藏在最后的幻灯片，反之亦然; 注意，只有当 infiniteLoop: false
speed integer 500 幻灯片滚动的速度，默认的单位为毫秒（MS）
easing String null 这个参数设置幻灯片切换的方式，如果使用 CSS transitions，包括一个值 transition-timing-function 属性。如果不使用 CSS transitions 你应该引入 plugins/jquery.easing.1.3.js 文件来配置切换方式

点击这里 http://gsgd.co.uk/sandbox/jquery/easing/ 了解更多信息。
slideMargin integer 0 每张幻灯片之间的间距
startSlide integer 0 开始的第一张幻灯片的索引，值从0开始计数
randomStart boolean false 是否开启随机切换幻灯片
captions boolean false 包括图片的说明文字，标题是来自图像的 title 属性。
ticker boolean false 在代码模式下使用滑块（类似于新闻）
tickerHover boolean false ticker 将暂停，当鼠标悬停在滑块。注：此功能不使用CSS过渡工作！
adaptiveHeight boolean false 根据每个滑块的高度动态调整滑块高度
adaptiveHeightSpeed integer 500 幻灯片高度转换持续时间（毫秒）,注：此功能adaptiveHeight为true！
video boolean false 支持视频，当设置为true时，需要 jquery.fitvids.js 支持
useCSS boolean true 如果true，将用 CSS transitions；如果false，将用jquery animate()
preloadImages String 'visible' 如果“all”，所有的图像开始之前，滑块预紧力。如果“visible”，只有在最初预加载图像可见幻灯片开始前的滑块（提示：使用“visible”如果所有的幻灯片都是相同的尺寸）, 可选：all, visible
responsive boolean true 启用或禁用滑块的自动调整大小。有用的，如果你需要使用固定宽度滑块。
slideZIndex integer 50 控制手柄的层级数
wrapperClass String 'lx-wrapper' slider最外层的类
fullScreenHeight integer 0 全屏时一定要设置最大高度
touchEnabled boolean true 是否允许触摸刷屏
swipeThreshold integer 50 为了执行一个slide渐变需要突出的像素数量 [只当touchEnabled为true时起作用]
oneToOneTouch boolean true 如果true，刷屏时无淡入淡出
preventDefaultSwipeX boolean true 如果true，横向刷屏
preventDefaultSwipeY boolean false 如果true，纵向刷屏
pager boolean false 设置是否显示分页，设置为true时，会在滑动内容下方显示分页图标
pagerType String 'full' 如果“full”，将为每个幻灯片生成一个链接。如果“short”，将使用一个X / Y（1 / 5）; 可选：full, short
pagerShortSeparator string ’/’ 分页链接下标的间隔(任意字符)
pagerSelector selector '' 定义分页控制元素
buildPager function(slideIndex) null 自定义分页内容。 详细例子： 

    $('.lxslider').lxSlider({
        buildPager: function(slideIndex){
            switch(slideIndex){
                case 0:
                    return '<img src="/images/thumbs/tree_root.jpg">';
                case 1:
                    return '<img src="/images/thumbs/houses.jpg">';
                case 2:
                    return '<img src="/images/thumbs/hill_fence.jpg">';
            }
        }
    });
pagerCustom selector null 分页内容是现成的DOM ，只要给定分页父级容器
iconfontPager boolean false 分页是否为iconfont字体生成
iconfontPagerUnicode Unicode '& #xe605;' 如果iconfontPager为true，分页内容就是此iconfont
controls boolean true 是否添加“上一个”、“下一个”按钮
nextText string 'Next' “下一个”按钮上面的文字
prevText string ’Prev’ “上一个”按钮上面的文字
nextSelector string null “下一个”按钮HTML
prevSelector string null “上一个”按钮HTML
autoControls boolean true 是否添加“开始停止”按钮
startText string 'Start' “开始”按钮上面的文字
stopText string ’Stop’ “停止”按钮上面的文字
autoControlsCombine boolean false 当幻灯片播放时只显示“停止”控件，反之亦然
controlsHover boolean false 如果是true，当鼠标经过时。。。
autoControlsSelector selector null 用于填充自动控件的元素
auto boolean false 是否自动渐变
pause integer 4000 每两个自动渐变之间的时间间隔
autoStart boolean true 是加载时还是点击start按钮时开始
autoDirection string ’next’ 自动滑动的方向 选：’next’, ‘prev’;
autoHover boolean false 鼠标放置是否暂停
autoDelay integer 0 自动延时
autoSlideForOnePage boolean false 鼠标放置后是否自动向后一张滑动
clickAutoPlay boolean false 点击暂停后是否自动向后一张滑动
minSlides integer 1 如果滑块数量比此值小，将按照从大到小排列
maxSlides integer 1 如果滑块数量比此值大，将按照从小到大排列
moveSlides integer 0 每次移动的滑块数量；如果为0，将默认为所有可视的滑块数量
slideWidth integer 0 每个滑块的宽度，水平滑动必须有

事件

事件 描述 实例
onSliderLoad 在滑块完全加载后立即执行 

    function(currentIndex){ // your code here }
onSlideBefore 在每个幻灯片转换之前立即执行。 

    function($slideElement, oldIndex, newIndex){ // your code here }
onSlideAfter 在每个幻灯片转换之后立即执行。 

    function($slideElement, oldIndex, newIndex){ // your code here }
onSlideNext 在每个“下一个”幻灯片转换之前立即执行。函数参数是目标（下一步）滑动元素。 

    function($slideElement, oldIndex, newIndex){ // your code here }
onSlidePrev 立即执行在每个“prev”幻灯片切换。函数的参数是目标（prev）滑动元件。 

    function($slideElement, oldIndex, newIndex){ // your code here }
onSliderResize 窗口大小发生变化时执行 

    function($slideElement, index){ // your code here }

方法

方法 描述 实例
goToSlide 跳转到第几个 

    slider.goToSlide(3);
goToNextSlide 跳转到下一个 

    slider.goToNextSlide();
goToPrevSlide 跳转到上一个 

    slider.goToPrevSlide();
startAuto 开始自动轮播 

    slider.startAuto();
stopAuto 停止自动轮播 

    slider.stopAuto();
getCurrentSlide 获取当前在第几个 

    var current = slider.getCurrentSlide();
getSlideCount 返回滑块中的幻灯片总数 

    var slideQty = slider.getSlideCount();
reloadSlider 重新加载滑块。在添加幻灯片时有用。接受可选设置对象。 

    slider.reloadSlider();
destroySlider 消毁当前slider 

    slider.destroySlider();
