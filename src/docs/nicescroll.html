自定义滚动条
内容区域 

    $('#demo-nicescroll').niceScroll({cursorborder:"",cursorcolor:"#cccdd1"});

    $('#demo-nicescroll').on('click',function(){
        $("#demo-nicescroll").getNiceScroll().resize();
    });

    $("#demo-nicescroll").on('mouseover',function() {
        $("#demo-nicescroll").getNiceScroll().resize();
    });

参数：

名称 默认值 描述
cursorcolor “＃000000” 十六进制改变光标颜色
cursoropacitymin 0 改变不透明度非常光标处于非活动状态（scrollabar“隐藏”状态），范围从1到0（隐藏）
cursoropacitymax 1 改变不透明度非常光标处于活动状态（scrollabar“可见”状态），范围从1（完全不透明） 到0
cursorwidth "5px" 像素光标的宽度
cursorborder "1px solid #fff" 游标边框css定义
cursorborderradius "5px" 以像素为光标边界半径
zIndex "auto" 改变z-index值的滚动条的div "number"
scrollspeed 60 滚动速度
mousescrollstep 40 高速滚动鼠标滚轮
touchbehavior false 使光标拖动滚动像在台式电脑触摸设备
hwacceleration true 使用硬件加速滚动支持的时候
boxzoom false 使变焦框中的内容
dblclickzoom true （仅当boxzoom = TRUE）变焦激活时，双击对话框
gesturezoom true （仅当boxzoom =真实，使用触摸设备）上缩放框激活时，间距输出/输入
grabcursorenabled true 显示“抢”图标的div touchbehavior = true时
autohidemode true 如何隐藏滚动条的效果， 可选值：true | "cursor" | false | "leave" | "hidden" | "scroll"
background '' 改变滚动条CSS背景
iframeautoresize true 在加载事件AUTORESIZE的iframe
cursorminheight 20 设置在像素的最小光标高度
preservenativescrolling true 您可以滚动本机可滚动区域用鼠标，冒泡鼠标滚轮事件
railoffset false 您可以添加抵消顶部/左边的轨道位置
bouncescroll false 使滚动弹跳在内容结尾作为移动像
spacebarenabled true 使向下翻页时，空格键已经按下滚动
railpadding { top: 0, right: 0, left: 0, bottom: 0 } 设置容器padding
disableoutline true 对于chrome浏览器，停用大纲（橙色hightlight）选择具有nicescroll一个div
horizrailenabled true nicescroll可以管理水平滚动
railalign right 取向垂直导轨
railvalign bottom 对齐水平导轨
enabletranslate3d true nicescroll可以使用CSS转换为滚动内容
enablemousewheel true nicescroll可以管理的鼠标滚轮事件
enablekeyboard true nicescroll可以管理键盘事件
smoothscroll true 滚动自如移动
sensitiverail true 点击轨道上进行滚动
enablemouselockapi true 可以用鼠标说明锁的API（对象拖动同样的问题）
cursorfixedheight false 用于光标在像素设置固定的高度
hidecursordelay 400 设置在微秒淡出滚动条的延迟时间
directionlockdeadzone 6 在对方向锁定激活像素死区
nativeparentscrolling true 检测内容底部，并让家长来滚动，作为原生滚动做
enablescrollonselection true 启用自动滚动的内容时
overflowx true
overflowy true
cursordragspeed 0.3
rtlmode "auto"
cursordragontouch false
oneaxismousemode "auto"
scriptpath getScriptPath()
preventmultitouchscrolling true 防止滚动触控事件

扩展：


    // 返回对象实例:
    var nice = false;
    $(document).ready(function() {
        nice = $("html").niceScroll();
    });

    // 改变光标的颜色:
    $(document).ready(function() {
        $("#thisdiv").niceScroll({cursorcolor:"#00F"});
    });

    // DIV with "wrapper", formed by two divs, the first is the vieport, the latter is the content:
    $(document).ready(function() {
        $("#viewportdiv").niceScroll("#wrapperdiv",{cursorcolor:"#00F"});
    });

    // 得到nicescroll对象:
    var nice = $("#mydiv").getNiceScroll();

    // 隐藏 nicescroll:
    $("#mydiv").getNiceScroll().hide();

    // 重新计算nicescroll高度:
    $("#mydiv").getNiceScroll().resize();

    // 滚动到某个位置 X , Y:
    $("#mydiv").getNiceScroll(0).doScrollLeft(x, duration); // Scroll X Axis
    $("#mydiv").getNiceScroll(0).doScrollTop(y, duration); // Scroll Y Axis
